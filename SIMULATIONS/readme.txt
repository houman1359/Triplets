Simulation_Pairwise_git.m :  a function to simulate data and compute the conditional information and interaction information on that data specified in the methods using both NPvC and GLM toolboxes provided in the folder NPvC. It saves the outputs containing these information values in the folder results.

Plot_simulations: plots the results generated by the Simulation_Pariwise_git.m code.

results foder: contains the results generated and saved by Simulation_Pariwise_git.m.
